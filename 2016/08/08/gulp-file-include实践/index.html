gulp-file-include实践

俺说句实话，在目前动态语言横行大道的时候，特别是php等语言都是可以将一些网站的公共部分提炼成common的时候，我这篇完全就是一篇填坑文章。

说下改的初衷。那还不是前一个都写成了静态html,使我这个后来人每次改动官网都很不爽。特别是公共的header、footer、aside等这种。每次一上个活动，都是想死的心都有。

原来每次交给下面的活大家都是改起来这么麻烦的，后来发现之，改之。

当然，就像我之前所说的，php,jsp哪怕是react等就不用往下看了。
